# disable-three-finger-gestures-redux
Disable the default three-finger touchpad gestures in GNOME

# Manual install
1. Run `./install.sh`
2. Then log out and back in. 
3. `gnome-extensions enable disable-three-finger-gestures-redux@cygnusx-1-org.github.com`
